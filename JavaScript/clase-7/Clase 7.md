## Secuencias de Escape

```sh
\' = Mostrar una comilla simple dentro de una cadena en la salida de informacion
\" = Mostrar una comilla soble dentro de una cadena en la salida de informacion
\\ = Barra invertida, diagonal inversa, es para mostrar una back slash
\n = Salto de linea
\r = Retorno
\t = Tabulacion
\b = Retroceso
\f = Salto de pagina

```

## Operador de Aumento

```sh
++
```
>EJEMPLO 1: 
```sh

```
>EJEMPLO 2:
```sh

```


## Operador de Decremento

```sh
--
```

>EJEMPLO 1: 
```sh

```
>EJEMPLO 2:
```sh
 z = a--; //PostDecremento
```
